#### pull 拉取镜像
docker pull ubuntu

#### 列出已下载镜像
docker image ls

#### 删除本地镜像

docker image rm - f  a2a15febcdf3



### 获取容器ip地址
`docker inspect --format='{{.Name}} - {{range.NetworkSettings.Networks}}{{.IPAddress}}{{end}}' $(docker ps -aq)`

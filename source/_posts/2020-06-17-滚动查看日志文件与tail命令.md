### 滚动实时查看nginx日志
```bash
tail -f access.log
```

### tail命令常用参数
access.log
```log
127.0.0.1 - - [17/Jun/2020:22:32:54 +0800] "GET / HTTP/1.1" 304 0 "-" "Mozilla/5.0 (Windows NT 10.0; ) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/83.0.4086.0 Safari/537.36"
127.0.0.1 - - [17/Jun/2020:23:03:41 +0800] "GET / HTTP/1.1" 304 0 "-" "Mozilla/5.0 (Windows NT 10.0; ) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/83.0.4086.0 Safari/537.36"
127.0.0.1 - - [17/Jun/2020:23:03:43 +0800] "GET / HTTP/1.1" 304 0 "-" "Mozilla/5.0 (Windows NT 10.0; ) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/83.0.4086.0 Safari/537.36"
```

```bash
// 默认显示10行内容
tail access.log

// -c 参数，加字节数 --bytes+NUM 按字节返回内容
tail -c10 access.log
//.36"
```
```bash
// -f 滚动实时查看nginx日志 -F --follow
tail -f access.log

// --pid=PID 与-f合用,表示在进程ID,PID死掉之后结束

// -n 显示最后NUM行 --lines = [+] NUM， 指定返回的行数。
tail -n3 access.log



```

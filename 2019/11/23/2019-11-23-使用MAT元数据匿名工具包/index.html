      什么是元数据Metadata?

      元数据是描述其它数据的数据（data about other data）， 比如一张图片是图像数据，那么图片的拍摄时间，拍摄地点等就是它的元数据。

    为什么要清除元数据？

    因为元数据包含时间，地点等个人敏感隐私数据。网上不乏有黑客通过社交照片的元数据信息追踪到用户地址，从而进行骚扰勒索等危险行动。包括此文中的博客, 上传图床前已经过脱敏处理.

Linux磁盘检测，分区与挂载
2019-09-23| Linux|字数总计: 810|阅读时长: 3 分钟
  • 新增1T磁盘一块

  1. 使用df -h查看当前硬盘信息（并未出现要挂在的磁盘）

  2. 使用命令fdisk -l查看所有磁盘信息（包括未挂载磁盘）

    磁盘 /dev/vdb 为需要挂载的磁盘。

  3. 通过lsblk查看

  4. 使用fdisk工具分区（当硬盘小于等于2T时，用fdisk，大于2T时，用parted）
    键入：fdisk /dev/vdb出现欢迎信息
    然后键入n新建分区

    p建立主分区， e建立扩展分区
    我们建立主分区，分区号，起始及结束扇区选择默认值：

    键入w写入分区并退出

  5. 再次使用lsblk查看

  6. 格式化为ext4文件系统
    mkfs.ext4 /dev/vdb1

  7. 使用mount临时挂载磁盘至/mnt目录
    mount /dev/vdb1 /mnt

  8. vim /etc/fstab更改配置文件永久挂载
    追加
    /dev/vdb1 /data ext4 defaults 0 0

  9. 使用 mount -a 判断加载是否正确
